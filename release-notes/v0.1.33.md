# AgentDock 0.1.33

## 中文

本版本提升状态刷新期间的界面响应速度，减少客户端检测和本地数据读取造成的等待。感谢 @xunjianxiang 提交 PR #2。

### 新增功能

- 客户端状态检测现在并行执行，并复用预先计算的命令搜索路径。
- 设置、桌面状态、供应商、Skills、MCP 和用量统计会通过并行任务加载。

### Bug 修复

- 修复刷新桌面状态时耗时的本地读取占用异步命令运行时、导致窗口短暂无响应的问题。
- 合并同一帧内重复的图标刷新请求，减少状态更新期间不必要的界面重绘。
- 清理并行检测改动引入的未使用函数警告，并补充显式搜索路径的客户端检测测试。

## English

This release keeps the interface responsive during status refreshes by reducing waits from client detection and local data reads. Thanks to @xunjianxiang for PR #2.

### New Features

- Client status detection now runs concurrently and reuses precomputed command search paths.
- Settings, desktop status, providers, Skills, MCP, and usage statistics are loaded through parallel tasks.

### Bug Fixes

- Fixed blocking local reads occupying the async command runtime and briefly freezing the window during desktop status refreshes.
- Coalesces duplicate icon refresh requests within the same frame to reduce unnecessary redraws during state updates.
- Removes unused-function warnings introduced by concurrent detection and adds coverage for client detection with explicit search paths.
