# AgentDock 0.1.22

## 中文

本版本修复 Antigravity 使用自定义供应商时仍要求登录、没有进入代理服务的问题。

### 新增功能

- 为 AgentDock 托管的 Gemini CLI 自动配置 API Key 认证方式。
- 启动 Antigravity 时自动修复旧安装缺失的认证配置，无需重新添加供应商或重新安装客户端。

### Bug 修复

- 修复 Antigravity 供应商连接测试成功，但启动客户端后仍显示登录界面的问题。
- 合并认证配置时保留 Gemini CLI 已有的主题、工具和其他用户设置。

## English

This release fixes Antigravity asking users to sign in and bypassing the configured proxy when a custom provider is active.

### New Features

- Automatically configures API-key authentication for the AgentDock-managed Gemini CLI.
- Repairs missing authentication settings on launch for existing Antigravity installations, without requiring users to recreate the provider or reinstall the client.

### Bug Fixes

- Fixed Antigravity showing the login flow after its custom provider passed the connection test.
- Preserves existing Gemini CLI themes, tools, and other user settings when merging the authentication configuration.
