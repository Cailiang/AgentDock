# AgentDock 0.1.35

## 中文

本版本更新 Antigravity 的 Gemini 默认模型，并改善供应商保存、切换和删除后的界面状态同步。

### 新增功能

- Antigravity 现在默认使用并优先推荐 `gemini-3.6-flash`，Google Official 预设、配置生成、模型回退列表和后端默认值保持一致。

### Bug 修复

- 修复保存或切换供应商后，界面必须等待完整后台重载才能显示最新激活状态的问题；供应商列表和当前配置现在会立即更新。
- 修复连续执行供应商操作时，较早完成的异步就绪检查可能覆盖最新状态的问题。
- 删除供应商后统一刷新客户端就绪状态，避免标题栏和供应商侧栏显示过期结果。

## English

This release updates the default Gemini model for Antigravity and improves UI state synchronization after saving, switching, or deleting providers.

### New Features

- Antigravity now defaults to and prioritizes `gemini-3.6-flash`, consistently across the Google Official preset, generated configuration, fallback model list, and backend defaults.

### Bug Fixes

- Fixed provider activation state waiting for a full background reload after saving or switching; the provider list and active configuration now update immediately.
- Fixed an older asynchronous readiness check potentially overwriting newer state during consecutive provider operations.
- Provider deletion now uses the shared readiness refresh path, preventing stale status in the header and provider sidebar.
